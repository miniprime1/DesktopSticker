## Overview

Desktop Sticker is a lightweight WPF application that brings your desktop to life. Pin your favorite animated characters (GIFs) or static images anywhere on your screen. It features a transparent, borderless window designed to be unobtrusive and resource-efficient.

Whether you want a cute companion watching over your work or a simple decorative sticker, this app handles it with modern Windows aesthetics.



## Getting Started

You can install Desktop Sticker _via_ provided installer.

### Installation

1. Get the latest installer from the [Releases Page](https://github.com/miniprime1/DesktopSticker/releases) and install with it.


> System Requirements: Windows 10/11 and .NET Framework 4.8.1.


## Build from Source

If you want to contribute or modify the code, follow these steps.

### Prerequisites

* [Visual Studio 2022](https://visualstudio.microsoft.com/vs/)
* .NET Framework 4.8.1 SDK

### Build using Visual Studio

1.  Clone the repository with following.
    ```bash
    git clone https://github.com/miniprime1/DesktopSticker.git
    cd DesktopSticker
    ```
2.  Open [`DesktopSticker.sln`](DesktopSticker.sln) in Visual Studio and build the solution
